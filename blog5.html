CARA KERJA INTERNET

Internet adalah jaringan global yang menghubungkan jutaan perangkat di seluruh dunia untuk saling bertukar data secara cepat dan otomatis.

Apa Itu Internet?

Internet adalah jaringan global yang menghubungkan komputer, server, smartphone, dan perangkat lainnya. Melalui internet, perangkat-perangkat ini dapat saling bertukar data seperti teks, gambar, video, dan suara dalam waktu sangat singkat.

Ketika kamu membuka sebuah website, sebenarnya perangkatmu sedang berkomunikasi dengan server yang berada di lokasi lain, bahkan bisa di negara yang berbeda.

DNS (Domain Name System)

Saat kamu mengetik alamat website seperti google.com, komputer tidak langsung mengerti. Internet menggunakan sistem DNS untuk mengubah nama domain tersebut menjadi alamat IP agar server tujuan dapat ditemukan.

DNS bekerja seperti buku telepon internet, yang mencocokkan nama website dengan alamat IP server.

Pengiriman Data (Packet)

Data yang dikirim melalui internet tidak dikirim sekaligus, melainkan dipecah menjadi bagian kecil yang disebut packet. Setiap packet bisa melewati jalur jaringan yang berbeda.

Setelah semua packet sampai di tujuan, perangkat akan menyusunnya kembali sehingga data dapat ditampilkan secara utuh.

Media Pengiriman Data

Internet menggunakan berbagai media pengiriman, salah satunya adalah fiber optik. Fiber optik mengirimkan data menggunakan cahaya dengan kecepatan sangat tinggi dan gangguan minimal.

Kabel fiber optik bawah laut menghubungkan internet antar negara dan benua.

Jaringan Seluler (BTS)

Internet pada smartphone bekerja melalui BTS (Base Transceiver Station). BTS menghubungkan perangkat ke jaringan operator menggunakan sinyal radio.

Semakin dekat dan baik kualitas BTS, semakin stabil koneksi internet.

WiFi dan Router

WiFi adalah jaringan nirkabel yang menghubungkan perangkat ke internet melalui router. Router berfungsi membagi koneksi internet ke banyak perangkat.

WiFi banyak digunakan di rumah, sekolah, dan tempat umum karena praktis.

Protokol TCP/IP

TCP/IP adalah protokol dasar internet. TCP memastikan data dikirim lengkap dan berurutan, sedangkan IP bertugas menentukan alamat tujuan data.

Kombinasi keduanya membuat komunikasi data di internet berjalan stabil dan akurat.

HTTP dan HTTPS

HTTP adalah protokol pertukaran data antara browser dan server. HTTPS adalah versi aman yang menggunakan enkripsi.

HTTPS melindungi data pengguna seperti password dan informasi pribadi dari penyadapan.
