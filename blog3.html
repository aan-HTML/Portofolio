Apa itu Coding?

Coding adalah proses menulis instruksi agar komputer dapat menjalankan perintah tertentu. Dengan coding, kita bisa membuat website, aplikasi, dan berbagai sistem digital.

Bagi pemula, coding sebaiknya dipelajari secara bertahap agar mudah dipahami dan tidak membingungkan.

HTML (Struktur Website)

HTML (HyperText Markup Language) adalah bahasa dasar yang digunakan untuk membuat struktur sebuah halaman website. HTML berfungsi untuk menyusun dan menampilkan konten, seperti judul, paragraf, gambar, tautan, tabel, dan tombol, agar dapat dibaca dan dipahami oleh browser.

Dalam HTML, setiap elemen ditulis menggunakan tag. Contohnya, tag <h1> digunakan untuk judul, <p> untuk paragraf, dan <img> untuk menampilkan gambar. HTML tidak mengatur tampilan atau desain, melainkan hanya menentukan isi dan susunan konten pada halaman web. 


<!DOCTYPE html>
<html>
  <head>
    <title>Website Pertama</title>
  </head>
  <body>
    <h1>Halo Dunia</h1>
    <p>Saya sedang belajar coding</p>
  </body>
</html>

CSS (Tampilan Website)

CSS adalah bahasa untuk mengatur tampilan halaman web. Kalau HTML menentukan struktur atau isi konten, CSS membuat konten itu terlihat menarik.

Dalam CSS, setiap elemen dapat diatur tampilannya menggunakan selector dan property. Misalnya: color untuk warna teks, background-color untuk warna latar, dan font-size untuk ukuran huruf. CSS membuat website lebih menarik tanpa mengubah isi dari HTML. 


body {
  background: black;
  color: white;
}

h1 {
  color: #ffc107;
}

JavaScript (Interaksi)

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat website menjadi interaktif. Dengan JavaScript, kita bisa membuat tombol bisa diklik, animasi berjalan, form bisa memvalidasi data, dan banyak lagi. Misalnya: alert("Halo Dunia!"); akan menampilkan pesan di layar saat dijalankan. JavaScript bekerja bersama HTML dan CSS untuk membuat halaman web dinamis dan menarik. 


alert("Halo, selamat belajar coding!");

document.querySelector("button")
  .addEventListener("click", () => {
    alert("Tombol diklik!");
  });

Dengan menguasai HTML, CSS, dan JavaScript, pemula sudah memiliki fondasi kuat dalam dunia web development.
